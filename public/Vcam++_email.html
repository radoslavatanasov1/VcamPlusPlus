VCam++ Setup Instructions

Welcome to VCam++!

Thank you for purchasing the VCam++ tweak! Please follow the instructions below to complete the installation and start using VCam++:

Installation Steps:

  1. Download the attached .deb file from this email.
  2. Use a sideloader app (e.g., Zebra or Sileo) to install the file on your device.
  3. If your device is RootFull:
    • Open MTerminal and type: killall SpringBoard.
  4. If your device is RootLess:
    • Restart SpringBoard after installation.
  5. Press + and - volume keys quickly to access the menu.
  6. Select a video from your gallery, and enjoy using VCam++!
